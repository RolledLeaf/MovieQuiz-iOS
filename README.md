## **MovieQuiz**

-Single screen quiz app about top 250 movies by IMDB. Built with Swift on Interface Builder

## Features
- Fetches movie data from a REST API
- Displays random questions about ratings
- MVP architecture
- Storyboard UI
- Unit and UI tests with XCTest

## Technologies
- Interface Builder
- URLSession
- MVP
- XCTest
